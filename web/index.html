OPML to Blogroll

Convert an OPML file of your RSS subscriptions into an HTML file to use as a blogroll.

To use:

  1. Export your RSS subscriptions as an OPML file
  2. Upload your OPML file
  3. Copy the generated HTML file and include it in your blog

Questions? Concerns? Please take a look at the FAQ.

By Leon Mika. Using simplecss and Flexoki.
